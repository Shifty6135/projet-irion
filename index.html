IRION Telechargement

Installe IRION en 1 minute

Recupere la derniere version de IRION . Le bouton Mise a jour dans IRION redirige vers cette page.

comment installer IRION

Telecharge la derniere version officielle depuis se site.

-> Telecharger IRION.exe

Si le navigateur bloque le fichier, ouvre les telechargements (Ctrl+J) puis clique sur Conserver.

Note: IRION ne demandera jamais d'informations personnelles. ou pouvent etre utilisées contre vous. Telechargez uniquement depuis les liens officiels Windows peut afficher une alerte SmartScreen pour les applications recentes. C'est normal. Verifiez que le fichier vient bien de ce depot GitHub puis cliquez sur Informations complementaires > Executer quand meme.

Comment mettre a jour IRION

  1. Ouvre IRION puis va dans Parametres.
  2. Clique sur Mettre a jour IRION.
  3. Clique sur Telecharger IRION.exe (GitHub).
  4. Ouvre IRION.exe puis accepte la creation du raccourci lors de la premiere ouverture.

Exemple d'ecran Windows

Si cet ecran apparait, c'est la protection Windows SmartScreen. Verifie la source puis clique sur Informations complementaires, ensuite Executer quand meme.
